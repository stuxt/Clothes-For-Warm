# Clothes-For-Warm

#衣路温暖
>最近的一则捐赠衣物成堆给受捐单位造成困扰的新闻，让我们对目前公益捐赠信息不对称引发了思考，因此准备开展此项目，能对现在的这种不对称现象起到改善的作用，让爱心变成爱，让衣服变成温暖。
>一个公益平台，这个平台旨在发布索捐信息和展示捐赠信息，实现捐赠信息对称。
